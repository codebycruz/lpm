# Installation

Install lpm using the command for your platform.

## Linux & macOS

```sh
curl -fsSL https://lualpm.com/install | sh
```

## Windows

```powershell
irm https://lualpm.com/install.ps1 | iex
```

## Verify

After installing, verify lpm is available:

```sh
lpm --version
# 0.6.0
```
