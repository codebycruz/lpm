# Installation

Install lpm using the command for your platform.

## Linux & macOS

```sh
curl -fsSL https://raw.githubusercontent.com/codebycruz/lpm/HEAD/install.sh | sh
```

## Windows

```powershell
irm https://raw.githubusercontent.com/codebycruz/lpm/HEAD/install.ps1 | iex
```

## Verify

After installing, verify lpm is available:

```sh
lpm --version
# 0.6.0
```
